# The 16-Bit, Single-Cycle Processor
## A Design for the Basys 3 FPGA Board

Using Xilinx Vivado and Verilog, our team will simulate a 16-bit processor and verify the design on the Basys 3 FPGA Board.

### Specifications
- Single-cycle, non-pipelined
- 16-bit word size
- Byte-addressable memory
- Instruction memory
    - Minimum size of 64 memory locations
- Data memory
    - Minimum size of 64 memory locations
- Register file
    - Sixteen 16-bit registers
- ALU
    - Has at least a zero flag
- Other required components:
    - Sign extender
    - Control unit
    - Multiplexer
    - Program counter
    - Adder
    - Shifter
